⚡ INICIO RÁPIDO (10 MINUTOS)
═══════════════════════════════════════════════════════════════════════════

1. INSTALAR (2 min)
   git clone <url>
   cd itgo
   npm install

2. CONFIGURAR (3 min)
   cp .env.example .env.local
   [Agregar credenciales de Supabase]

3. EJECUTAR (2 min)
   npm run dev
   Abrir http://localhost:3000

4. LOGIN (1 min)
   Email: admin@itgo.com
   Password: password

5. DESPLEGAR EN VERCEL (2 min)
   vercel deploy --prod
